# This file contains cron job configurations for the Hacker News project

# To view current cron jobs, run: crontab -l
# To edit cron jobs, run: crontab -e
#
# Cron format: MIN HOUR DAY MONTH WEEKDAY COMMAND
# Examples:
# 0 */4 * * * - Every 4 hours at minute 0
# 0 0,4,8,12,16,20 * * * - At 12am, 4am, 8am, 12pm, 4pm, 8pm
